# **App Name**: Novelku

## Core Features:

- Genre Browsing: Browse web novels by genre categories. All text in Bahasa Indonesia.
- Chapter Reading: Read individual chapters of a web novel with a clean, immersive reading interface. All text in Bahasa Indonesia.
- Coin System: Manage a coin balance and use coins to unlock access to new chapters.
- Personal Library: Save and manage ongoing reads in a personal library. All text in Bahasa Indonesia.

## Style Guidelines:

- Primary color: Dark, muted purple (#4A335C) to set a somber and magical tone.
- Background color: Very dark gray (#121212) for an immersive, distraction-free reading experience.
- Accent color: Subtle gold (#A68B5B) for highlights and interactive elements, conveying sophistication.
- Body and headline font: 'Literata', a transitional serif font for a literary feel.
- Use minimalist icons with a gold or light gray fill for key actions like adding to library, unlocking chapters.
- Use a clean, single-column layout for chapter reading to focus on the text, with unobtrusive navigation.
- Subtle transition animations between chapters, and loading animations with the gold accent color to enhance immersion.
